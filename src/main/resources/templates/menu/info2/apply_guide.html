  • 수강신청

  • 결제

  • 수강신청완료

증빙자료를 평생교육원 교학팀(대학원동 506호)에 제출해주셔야 장학 및 감면 혜택이 적용됩니다.(제출서류 : 장학제도 및 수강료 감면기준 참고)

접수방법

일반교육과정
  • 1. 전화접수

    • - 전화 :

    ※ 전화 접수는 기존 수강생만 가능합니다.

  • 2. 방문접수 : 단국대학교 평생교육원 교학팀(대학원동 506호)
  • 3. 인터넷접수

    • - 칼빈대학교 평생교육원 홈페이지 상단의[로그인/회원가입]메뉴 클릭

    - 기존 수강생일 경우 로그인 : 이메일, 비밀번호로 로그인

    - 신규 수강생일 경우 회원가입 : 이메일 주소로 회원가입

    - 수강신청을 원하는 강의의 페이지로 이동후 수강신청

수강료 입금

- 카드결제가 되지 않으며, 현금영수증 발행되지 않습니다.
- 입금시 수강생의 성명기재를 하며, 입금 후 반드시 전화로 등록을 확인하시기 바랍니다.
